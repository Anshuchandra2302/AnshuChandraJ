## What Issues You Ran Into and How You Solved Them

### 1. Claude Code Subscription / Authentication Wall
*   **Issue:** When attempting to log into the Claude Code add-on, the authentication process failed or blocked progress because the tool required an active paid subscription or specific API tier access, making a standard free login impossible.
*   **Solution:** Documented the constraint as an environmental limitation. Shifted focus to completing the rest of the workspace setup using Cursor's native AI features and the successfully authenticated Codex extension to manage the project workflow.

### 2. Git Integration and Repository Connection Failure
*   **Issue:** Cursor was initially unable to recognize, pull, or manage the GitHub repository because Git was not installed on the local machine, and the IDE lacked permission to interact with the GitHub account.
*   **Solution:** Downloaded and installed Git manually on the system. Then, enabled Git version control within the Cursor settings, securely connected the GitHub account directly to the Cursor IDE interface, and successfully cloned the repository into the local workspace.
